# 🏛️ GDG Campus Hub

> An open-source Flutter project by GDG — learn Flutter and app architecture by contributing to a real app.

**Primary goal:** Understand Flutter (state management, navigation, clean architecture) by building.  
**Secondary goal:** A working campus event hub for students.

## Tech Stack
`Flutter` · `Dart` · `Riverpod` · `GoRouter` · `Dio` · `Freezed` · `flutter_secure_storage` · `GetIt`

## Getting Started

```bash
git clone https://github.com/BhairabMahanta/gdg-campus-hub.git
cd campus-hub
flutter pub get
flutter run
```

Supports: Android, iOS, Web
